# Ground Black - Интернет-магазин кофе

## Описание проекта

Ground Black - это полнофункциональный интернет-магазин, специализирующийся на
продаже кофе, аксессуаров для кофе и шоколадных добавок. Проект включает в себя
каталог товаров, систему корзины и заказов, блог с статьями о кофе,
пользовательскую систему с авторизацией и админ-панель для управления контентом.

## Технологии

- **Backend**: PHP 7.4+
- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.3.3
- **База данных**: MySQL 5.7+
- **JavaScript библиотеки**:
  - Splide.js 4.1.4 (для каруселей)
  - Bootstrap Icons 1.11.1
- **Стилизация**: SCSS (с компиляцией в CSS)
- **Шрифты**: Charter (OTF/TTF)

## Функциональность

### Для пользователей:

- Просмотр каталога товаров с фильтрами по цене, категориям
- Детальная информация о товарах с изображениями
- Корзина покупок с возможностью добавления/удаления товаров
- Оформление заказов с выбором доставки и оплаты
- Система избранного
- Регистрация и авторизация
- Блог с статьями о кофе
- Контактная форма

### Для администраторов:

- Управление товарами (добавление, редактирование, удаление)
- Управление категориями
- Управление статьями блога
- Просмотр и изменение статуса заказов
- Управление пользователями

## База данных

Проект использует MySQL базу данных с следующими основными таблицами:

- `users` - Пользователи
- `items` - Товары
- `items_images` - Изображения товаров
- `categories` - Категории товаров
- `articles` - Статьи блога
- `order-info` - Заказы
- `contact-us` - Контактные формы
- `promo` - Промокоды
- `roles` - Роли пользователей
- `status` - Статусы заказов

## Установка и запуск

### Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше
- Веб-сервер (Apache/Nginx)
- Node.js и npm (для управления зависимостями)

### Шаги установки

1. **Клонируйте репозиторий:**

   ```bash
   git clone <repository-url>
   cd coffee-shop
   ```

2. **Установите зависимости:**

   ```bash
   npm install
   ```

3. **Настройте базу данных:**
   - Создайте базу данных MySQL
   - Импортируйте дамп из файла `ground-black (1).sql`
   - Обновите настройки подключения в `modules/db.php`:
     ```php
     $servername = "localhost";
     $username = "your_username";
     $password = "your_password";
     $database = "your_database_name";
     ```

4. **Настройте веб-сервер:**
   - Убедитесь, что корневая директория указывает на папку проекта
   - Включите mod_rewrite для Apache (если используется)

5. **Запустите проект:**
   - Откройте браузер и перейдите по адресу вашего локального сервера

## Использование

### Для пользователей:

1. Зарегистрируйтесь или войдите в систему
2. Просматривайте каталог товаров
3. Добавляйте товары в корзину
4. Оформляйте заказы
5. Читайте статьи в блоге

### Для администраторов:

1. Войдите под учетной записью администратора
2. Перейдите в админ-панель через личный кабинет
3. Управляйте товарами, категориями и заказами
